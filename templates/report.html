Abstract

In this report, we delve into the practical use and examina- tion of various geometric algorithms. These algorithms are specifically designed for two fundamental tasks: constructing convex hulls and detecting intersections between lines. The construction of convex hulls involves finding the smallest pos- sible convex shape that encloses a given set of points. Simul- taneously, the detection of line intersections focuses on iden- tifying points where two or more lines cross each other. By analyzing the implementation and outcomes of these geomet- ric algorithms, this report aims to provide insights into their efficiency and applicability in solving real-world problems.

Background

Convex Hull Problem

The Convex Hull problem is a fundamental concept in com- putational geometry. Given a set of points in the Euclidean plane, the convex hull is the smallest convex polygon that en- closes all the points. Formally, it is the intersection of all convex sets containing the given points. Convex hulls have widespread applications in areas such as computer graphics, pattern recognition, and robotics.

The problem can be formally stated as follows: Given a set of points P = (x1, y1), (x2, y2), . . . , (xn, yn) , find the convex hull, denoted as CH(P ), which is the convex polygon formed by the subset of points.

Various algorithms exist to compute the convex hull, such as Graham's scan, Jarvis march, and QuickHull. These al- gorithms differ in their efficiency and approach but share the common goal of determining the convex hull in a time com- plexity that is optimal or near-optimal.

Line Intersection Problem

The Line Intersection problem involves determining if and where two or more line segments intersect in the Euclidean plane. This problem is crucial in computer graphics, geo- graphic information systems, and computational geometry.

Given a set of line segments L = l1, l2, . . . , lm , where each line segment is defined by two endpoints, the goal is to identify all intersections between these line segments. An intersection occurs when two line segments share a common point.

{ }

The Line Intersection problem has applications in various fields, including robotics path planning, computer-aided de- sign, and computational biology.

Efficient algorithms, such as the sweep line algorithm and Brute Force, have been developed to address the Line Inter- section problem. These algorithms aim to provide accurate and fast solutions for identifying intersecting line segments within a given set.
